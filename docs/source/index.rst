Relectric Systems
=================

About
-----

Source code behind the development of the system architecture for our
first electric car conversion. This code will allow us to connect our
`User
Interface <https://github.com/relectric-car-team/user-interface>`__ to
our hardware interfaces such as
`CANBUS <https://github.com/relectric-car-team/canbus-mcu-base>`__.

Contributing
------------

Please use semantic commit messages and branch naming conventions using
this guide. Private branches should be named using the
semantic/name/purpose convention. For example: docs/ratik/update-readme
signifies that Ratik is responsible for this documentation change and
the purpose of the branch is to update the README. Please base all pull
requests off of the main branch as they will be rebase merged. The
linear history requirement is enforced on main.
